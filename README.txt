NAKİT AKIŞ YÖNETİM BİLGİ SİSTEMİ - FİNAL SİTE PAKETİ

Yükleme:
1) ZIP içeriğini hosting/public_html veya site ana dizinine çıkarın.
2) Ana dosya: index.html
3) Logo ve favicon dosyaları: assets/ klasöründe.
4) Güncel Excel verinizden üretilen data.json dosyasını aynı dizine atabilirsiniz.

Notlar:
- index.html tek başına çalışır; içinde mevcut veri gömülüdür.
- Daha sonra data.json güncellemek isterseniz panelde "Veri Güncelle" bölümünden data.json seçebilirsiniz.
- Chart.js CDN üzerinden yüklenir; site internete açıksa grafikler çalışır.
- Pro logo: assets/logo-pro.png
- Faviconlar tüm modern tarayıcılar için assets klasöründe hazırdır.
